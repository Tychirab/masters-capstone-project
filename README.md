# Masters Capstone Project
### Designing a Healthcare Information System for Clayton State University Health Services

I completed a comprehensive system design project focused on improving access to student immunization information at Clayton State University. The project explored how the university, Atrium Health, and the Georgia Registry of Immunization Transactions and Services (GRITS) could share data more efficiently with student consent.

The current process requires students to visit the Atrium Health office in person to authorize record retrieval. With no electronic consent option and no system integration across networks, staff must manually search for records, which creates delays during enrollment, housing assignments, and clinical services. My project examined these gaps and proposed a new Healthcare Information System that supports secure, automated, and user-friendly data sharing.

Using qualitative research, I interviewed Atrium Health staff and reviewed existing documentation to identify operational challenges, privacy concerns, and desired system features. I then translated these insights into functional and nonfunctional requirements. Key needs included electronic consent, real-time immunization retrieval from GRITS, a centralized dashboard, role-based access, and detailed audit logs.

To illustrate system behavior, I created a full set of system design artifacts, including use case diagrams, sequence diagrams, a class diagram, window navigation flows, and CRC cards. The proposed system supports secure data exchange using HL7 or FHIR standards and emphasizes privacy, interoperability, and user-centered design.

This project demonstrates my ability to evaluate healthcare workflows, analyze information systems, and design technical solutions that improve efficiency and access to care. It also reflects my commitment to supporting student well-being through better health information management.
